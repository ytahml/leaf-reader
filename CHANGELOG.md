# Changelog

## [0.1.0] - 2026-05-31

### ✨ 功能

- 支持打开单个 `.md` 文件阅读
- 支持打开文件夹浏览 Markdown 文件树
- Markdown 渲染 + 代码语法高亮
- 三套阅读主题：晨光 / 夜读 / 青禾
- 阅读元信息：预估阅读时间、文件日期
- 顶部滚动进度条
- 自定义无边框标题栏（macOS 风格）
- 键盘快捷键支持
- 主题设置持久化

### 🛠️ 技术

- Tauri v2 + Vue 3 + TypeScript
- Pinia 状态管理
- markdown-it + highlight.js
- CSS Custom Properties 主题系统
